Welcome to RESTFul Fusion 1.0.1

Purpose

This is a demo application which demonstrates how one can build a general purpose REST Interface for Oracle Fusion CRM, aka Oracle Sales Cloud. It is an extension of a blog article https://blogs.oracle.com/angelo/entry/rest_enabling_oracle_fusion_sales.

 

Object Support

  • Opportunities
  • Locations
  • Person
  • SalesParty
  • SalesLead
  • Interaction
  • Resource
  • UserDetails
